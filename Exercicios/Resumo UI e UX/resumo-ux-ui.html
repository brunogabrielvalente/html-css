Resumo UX, UI, Front-end e Back-end

🧠 UX (User Experience)

Planeja a experiência do usuário. Define como o site ou app deve funcionar para ser fácil e agradável de usar.

🎨 UI (User Interface)

Cuida da aparência: cores, botões, ícones, tipografia e layout. Deixa tudo bonito e organizado.

💻 Front-end

Transforma o design em código (HTML, CSS e JavaScript). É o que o usuário realmente vê e interage.

⚙️ Back-end

Trabalha por trás do site: banco de dados, autenticação, servidores e lógica. Faz tudo funcionar corretamente.

💡 UX e UI estão ligados ao front-end, enquanto o back-end é o cérebro por trás da aplicação.
